# Nuvreon Landing Page Images

Place your images in this directory. The following filenames are referenced by the landing page components:

## Required Images

| Filename | Section | Recommended Size | Description |
|----------|---------|-----------------|-------------|
| `hero-bg.jpg` | Hero | 1920x1080 | Hero background image (optional, overlaid with dark gradient) |
| `about.jpg` | About | 800x600 | Company / team photo or office image |
| `platform-01.jpg` | Market Intelligence Platform | 600x400 | Digital analytics / dashboard visual |
| `platform-02.jpg` | Influence Intelligence SaaS | 600x400 | Influencer / social media visual |
| `platform-03.jpg` | Operational Intelligence Software | 600x400 | Operational systems / infrastructure visual |
| `vision.jpg` | Vision | 800x600 | Adaptive environments / device ecosystem visual |
| `logo.svg` | Navigation | any | Company logo (optional, text logo used as fallback) |

## Notes
- All images are optional. The page displays gracefully without them.
- Use `.jpg`, `.png`, or `.webp` formats for photos.
- Use `.svg` for logos and icons.
- Recommended: optimize images before adding (compress to <500KB each).
